README

---------------------------------------------------------------------------------------------------------
1. 테서렉트 설치
https://github.com/UB-Mannheim/tesseract/wiki


---------------------------------------------------------------------------------------------------------
2. 환경변수 설정
시스템 속성 -> 고급 -> 환경 변수 -> Tesseract-OCR 설치한 폴더 추가


---------------------------------------------------------------------------------------------------------
3. 파이참 터미널창에서 설치

pip install numpy
pip install opencv-python
pip install matplotlib
pip install textblob
pip install autocorrect


---------------------------------------------------------------------------------------------------------
4. 파이참 86~88 줄 파일 경로 지정

테서렉트가 저장된 경로 
pytesseract.pytesseract.tesseract_cmd = r'C:\Program Files\Tesseract-OCR\tesseract.exe'
이미지를 읽어올 폴더
path_dir = 'C:\\Users\\poor1\\Desktop\\scan_folder'
이미지를 저장하고 글자를 추출하여 txt로 저장할 폴더
save_dir = 'C:\\Users\\poor1\\Desktop\\filtered_image_save'

디폴트 path_dir, save_dir 사용예)
1. poor1대신 사용자 이름입력
2. 바탕화면에 scan_folder, filtered_image_save 폴더 생성


---------------------------------------------------------------------------------------------------------
5. path_dir 에 있는 폴더에 글자를 추출할 이미지를 넣고 파이참 실행 
(여러 이미지 가능)
(실시간으로 필터링되는 이미지를 save_dir 에 있는 폴더에서 확인 가능)


---------------------------------------------------------------------------------------------------------
6. 파이참 중간에 각각의 이미지마다 사용된 필터와 최종 필터링된 이미지 확인. 
(이미지마다 출력) 

실행 완료후 save_dir 폴더 내에 최종 필터링된 이미지와 추출된 글자 확인
